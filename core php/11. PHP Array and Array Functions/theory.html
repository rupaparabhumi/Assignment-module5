Ans11:Define arrays in PHP. What are the different types of arrays?

In PHP, there are three types of arrays:

  • Indexed arrays - Arrays with a numeric index.
  • Associative arrays - Arrays with named keys.
  • Multidimensional arrays - Arrays containing one or more arrays.

    • indexed Array:

  • These are the most common and regular array types used in PHP, where
    the values have numeric indexes by which they can be referenced.

    • Associative Array:

  • With these arrays, the stored values are given name keys to serve as an index instead of
    the numeric index.They are mainly used when fetching named columns of a database table.

    • Multidimensional Array:

  • These types of arrays are usually known as arrays of arrays (2d) or arrays of arrays of arrays (3d),
    and so on. They are simply arrays that contain other arrays.
