    for loop

  • The iteration is clearly visible.The block of code is repeated as long the condition is met or the counter meets a specific value.
  • Good performance.
  • The stop condition is specified easily.
  • upon working with collection,it needs the usage of the count() function

    • foreach loop

  • The iteration is hidden.The block of is repeated until iterating over the array is complete better performance.
  • The stop condition has to be explicitly specified.
  • it can simply work without the usage of the count()method.
